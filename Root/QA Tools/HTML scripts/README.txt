To install this script environment is required to download nvm-node version manager nad npm-node package manager. They are available in Lion Store.

Next step - how to set environment:
 - open powershell/cmd
 - go to directory with script
 - write "npm i" - all packages will be installed
 - ready to use

How to use:
 - src directory is storage of document - ther should be just 1 "*.html" file and one folder "images" (you can create a label for "src" folder and move it in workspace)
 - open powershell/cmd and go to directory with script
 - write command "npm start"
